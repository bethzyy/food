# ✅ 天干地支显示已调整为靠左对齐!

## 🔧 修改内容

### 问题分析

之前天干地支在节气日期间会居中显示,这不符合靠左对齐的要求。

**原因**: `style.css:1476` 行有 `text-align: center` 覆盖了基础样式

### 解决方案

修改了 `style.css:1476` 行:
```css
/* 修改前 */
body.is-solar-term-day .ganzhi-display {
    text-align: center;  /* ❌ 居中 */
}

/* 修改后 */
body.is-solar-term-day .ganzhi-display {
    text-align: left;    /* ✅ 靠左 */
}
```
