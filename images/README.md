## 文件命名规范

**非常重要**：文件名必须与应用中的节气/节日名称完全匹配！

- 24节气：使用完整的节气名称（例如：`立春.png`）
- 传统节日：使用完整的节日名称（例如：`春节.png`、`元宵节.png`）
- 文件格式：PNG格式（支持透明背景）

## 使用方法

1. 应用自动根据当前节气/节日查找对应的PNG文件
2. 如果本地文件存在，优先使用本地插画
3. 如果本地文件不存在，自动降级到CSS渐变背景

## 添加新插画

1. 打开 `illustration_download_guide.html` 查看下载指南
2. 在百度图片搜索对应的关键词（例如："立春插画 中国风"）
3. 下载中国风插画（推荐插画，避免照片）
4. 保存为PNG格式到 `images/festival_art/` 目录
5. **注意**：文件名必须与应用中的名称完全匹配

## 自动降级机制

应用实现了智能的背景加载系统：

```
1. 尝试加载本地PNG文件
   ↓ (失败)
2. 自动使用CSS渐变背景
   ↓ (24节气渐变或传统节日渐变)
3. 应用70%白色雾化遮罩
```

## 图片格式建议

- **格式**: PNG（支持透明背景）
- **分辨率**: 建议1920x1080或更高
- **风格**: 中国风插画、水墨画、工笔画
- **避免**: 照片、卡通风格、低分辨率图片

## 雾化效果说明

所有背景图都会自动应用70%白色半透明遮罩（`rgba(255, 255, 255, 0.7)`），确保：

- ✅ 文字清晰可读
- ✅ 背景插画隐约可见
- ✅ 整体视觉效果柔和美观

可在 `style.css` 中调整雾化强度：
```css
.app-container::before {
    background: rgba(255, 255, 255, 0.7);  /* 调整这个值 */
}
```
