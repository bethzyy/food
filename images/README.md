# 节气和传统节日背景图

本目录集中管理所有节气和传统节日的背景图URL。

## 文件说明

### `background_images.json`
包含所有24节气和14个传统节日的背景图URL配置。

- **solar_terms**: 24节气背景图（中国风水墨画、山水画）
- **festivals**: 传统节日背景图（春节、元宵、清明、端午、七夕、中秋、重阳、腊八、小年、除夕、祭灶、寒食、上巳、冬至）

## 图片来源

所有图片来自维基媒体公共领域，主要是：
- 宋代山水画（李唐、夏圭、米芾等）
- 元代山水画（黄公望、倪瓒、王蒙等）
- 明代山水画（沈周、文徵明、唐寅、董其昌等）
- 清代山水画（王翚、王鉴、龚贤等）

## 特点

1. **中国风风格**：全部采用传统水墨画、山水画
2. **公共领域**：无版权问题，可自由使用
3. **高分辨率**：1920px宽度，适合做背景
4. **稳定可靠**：维基媒体CDN，加载稳定

## 使用方法

在主程序`app.js`中，`setSolarTermBackground`方法会根据节气或节日名称自动加载对应的中国风背景图。

## 修改指南

如需更换背景图，只需修改`background_images.json`中对应的URL即可。
