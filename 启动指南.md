# 快速启动指南

## 最简单的启动方式

### 🎯 方法1：一键启动（推荐）

直接双击运行：**`启动.bat`**

这个脚本会：
- ✅ 自动检测Python
- ✅ 自动检测环境变量
- ✅ 自动启动正确的服务器
- ✅ 自动打开浏览器

---

## 如果一键启动失败

### 方法2：使用Python命令

**1. 打开命令行（CMD或PowerShell）**

**2. 进入应用目录**
```bash
cd C:\D\CAIE_tool\MyAIProduct\food
```

**3. 启动服务器**
```bash
python server_with_env.py
```

或者简单版：
```bash
python start_server.py
```

**4. 浏览器访问**
http://localhost:8000

## 首次使用

### 1. 获取API Key

访问智谱AI官网：https://open.bigmodel.cn/

注册并登录后，在控制台获取API Key

格式类似：`1234.abc123def456789`

### 2. 设置API Key（两种方式）

**方式A：设置环境变量（推荐，只需设置一次）**

Windows PowerShell：
```bash
$env:ZHIPU_API_KEY="你的API Key"
```

Windows CMD：
```bash
set ZHIPU_API_KEY=你的API Key
```

然后运行：`python server_with_env.py`

**方式B：浏览器中输入（无需设置）**

直接运行：`python start_server.py` 或双击 `启动.bat`

首次使用时，浏览器会弹出输入框，输入API Key即可（会自动保存）

---

## 停止服务器

在命令行窗口按 `Ctrl + C`

## 常见问题

### Q: 提示"Python未安装"
A: 请先安装Python 3.7+
下载地址：https://www.python.org/downloads/

### Q: 端口8000被占用
A: 修改端口号，例如使用9000：
```bash
python -m http.server 9000
```
然后访问 http://localhost:9000

### Q: 浏览器没自动打开
A: 手动在浏览器地址栏输入：http://localhost:8000

### Q: API Key在哪里输入
A: 如果设置了环境变量会自动读取，否则首次点击"生成推荐"时会提示输入
