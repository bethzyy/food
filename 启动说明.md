# 启动说明

## 问题解决：批处理文件乱码

如果看到乱码错误，请使用以下启动方式：

### 方法1：使用 Python 脚本（推荐）

```bash
python start_server.py
```

这个脚本会：
- 自动启动 HTTP 服务器
- 自动打开浏览器
- 显示友好的中文提示

### 方法2：使用简化的批处理文件

```bash
start.bat
```

### 方法3：手动启动（最可靠）

```bash
cd C:\D\CAIE_tool\MyAIProduct\food
python -m http.server 8000
```

然后在浏览器中访问：http://localhost:8000

### 方法4：使用其他 HTTP 服务器

**如果有 Node.js：**
```bash
npx http-server -p 8000
```

**如果有 PHP：**
```bash
php -S localhost:8000
```

## 访问应用

服务器启动后，在浏览器中打开：

**主应用：** http://localhost:8000/index.html

**测试页面：** http://localhost:8000/tests/test_sample.html

## 首次使用

1. 打开应用后会要求输入 API Key
2. 获取 API Key：https://open.bigmodel.cn/
3. 格式：`id.secret`（例如：`1234.abc123def456`）
4. API Key 会保存在浏览器中，无需重复输入

## 停止服务器

在命令行窗口按 `Ctrl + C`

## 常见问题

### Q: 端口 8000 被占用
A: 使用其他端口，例如：
```bash
python -m http.server 9000
```

### Q: Python 未安装
A: 请先安装 Python 3.7+：https://www.python.org/downloads/

### Q: 浏览器无法打开
A: 手动在浏览器地址栏输入：http://localhost:8000

## 推荐浏览器

- Chrome（推荐）
- Edge
- Firefox
- Safari

需要支持现代 JavaScript 特性（ES6+）
